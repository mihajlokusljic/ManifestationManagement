Adding label

When adding label, user can enter following data:

  • Label Id - Value of label identifier (id) must be unique and can be manually entered. Also, there is an option for automatic generation of id by selecting "Auto-generated Ids" field. If id is entered manually there is length constraint: from 2 to 60 characters.
  • Label Color - Visual representation of label, by adding color to it. Must be picked.
  • Label Description - More detailed description about label. It has length constraint with minimum 2 characters.

By clicking on button "Add label" user adds label to system. In case of clicking on button "Cancel" adding label is cancelled and user is redirected to window that shows current label in system.
