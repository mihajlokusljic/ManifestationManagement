Manifestation Managment App

About app

This app is created for managing manifestation purposes. That means you can create manifestation , put it anywhere you want all around Novi Sad, edit or delete, you can also list all of your created manifestations. Main elements of this app are:

  • Label - they are used for labeling manifestation, manifestation can have more than one label, for them you can pick color, id and description
  • Manifestation Type - they are used for setting type of manifestation, manifestation can have only one type, for them you can pick icon, name, id and description
  • Manifestation - they are used for defining manifestation, for them you can pick id, name, labels, type, icon (optionally, if not setted, get icon from manifestation type) , is it outside, description and more

Drag and drop

Drag and drop method is simple way where user can with ease put his manifestation that user created anywhere on the map of Novi Sad with dragging it from list of all manifestation and dropping it on the map. Also user can edit manifestation location on the same way.

Pages with shortcuts

  • CTRL+D   -  Add Label
  • CTRL+F   -  Show Labels
  • CTRL+E   -  Add Manifestation Type
  • CTRL+R   -  Show Manifestation Types
  • CTRL+Q   -  Add Manifestation
  • CTRL+W   -  Show Manifestations
  • CTRL+F1  -  Show Novi Sad Map
  • CTRL+F2  -  Show Stari Grad Map
  • CTRL+F3  -  Show Detelinara Map
  • CTRL+F4  -  Show Liman Map
  • F1       -  Showing Help Documentation
