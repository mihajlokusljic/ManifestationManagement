Manifestations preview

This page displays all added manifestations. The manifestation are shown within a table containing most important manifestation features. For each manifestation you can see its icon, id, name of its type and its date. You can delete or edit a manifestation by selecting it in the table and clicking the "Edit Manifestation" button or "Delete Manifestation" button at the bottom of the form. Also you can easily find specific manifestations by typing its id or name in the search and filter fields.

Edit a manifestation

For more details about a manifestation you can select a manifestation in the table and click the "Edit Manifestation" button to enter the edit form, where you can see and change values of all attributes for the selected manifestation. The form displays the current data for the selected manifestation that can be changed. If you want to save the changes click the "Confirm changes" button. If you make a mistake that you want to undo just click the "Cancel" button at the bottom of the edit page. After confirming or canceling the changes you will be returned to the manifestations privew where you can see the resulting data for the edited manifestation in the table.

Delete a manifestation

You can delete a manifestation by selecting it in the table and clicking the "Delete Manifestation" button. Note that this is a destructive operation that can not be undone. When deleting a manifestation you will be prompted with a warning and asked to confirm the delete operation. If you choose "Yes" in the warning dialog the manifestation will be permanently deleted and it will no longer be displayed on the maps.

Searching manifestations

You can easily distinguish certain manifestations by doing a search. Enter a part or full manifestation id or name in the "Search" field at the top left part of the page and press Enter or click the search button. All manifestations that contain the given input in their id or name will be highlighted with red color in the table. The search is case sensitive.

Filtering manifestations

You can easily access certain manifestations by performing a filter. Enter a part or full manifestation id or name in the filter field at the top right part of the page and press Enter or click the "Filter" button. Only the manifestations that contain the given input in their id or name will be displayed in the table. Filtering is case sensitive.
