Adding manifestation

When adding manifestation, user can enter following data:

  • Label - they are used for labeling manifestation, manifestation can have more than one label, for them you can choose color, id and description
  • Manifestation Type - they are used for setting type of manifestation, manifestation can have only one type, for them you can choose icon, name, id and description
  • Manifestation - they are used for defining manifestation, for them you can choose id, name, labels, type, icon (optionally, if not setted, default icon is set from manifestation type) , is it in outside environment, description and more
