Manifestation types preview

This page displays all added manifestation types. The manifestation types are shown within a table. For each manifestation type you can see its icon, id, name and description. You can delete or edit a manifestation type by selecting it in the table and clicking the "Edit Type" button or the "Delete Type" button at the bottom of the page. Also you can easily find specific manifestation types by typing its id or name in the search and filter fields.

Edit a manifestation type

For more details about a manifestation type you can select a manifestation type in the table and click the "Edit Type" button to enter the edit form, where you can see and change values of all attributes for the selected manifestation type. The form displays the current data for the selected manifestation type that can be changed. If you want to save the changes click the "Confirm changes" button. If you make a mistake that you want to undo just click the "Cancel" button at the bottom of the edit page. After confirming or canceling the changes you will be returned to the manifestation types privew where you can see the resulting data for the edited manifestation type in the table.

Delete a manifestation type

You can delete a manifestation type by selecting it in the table and clicking the "Delete Type" button at the bottom of the page. A manifestation type can be deleted only if there are no manifestations marked with the given type. Note that this is a destructive operation that can not be undone. When deleting a manifestation type you will be prompted with a warning and asked to confirm the delete operation. If you choose "Yes" in the warning dialog the manifestation type will be permanently deleted.

Searching manifestation types

You can easily distinguish certain manifestation types by doing a search. Enter a part or full manifestation id or name in the "Search" field at the top left part of the page and press Enter or click the search button. All manifestation types that contain the given input in their id or name will be highlighted with red color in the table. The search is case sensitive.

Filtering manifestation types

You can easily access certain manifestation types by performing a filter. Enter a part or full manifestation type id or name in the filter fild at the top right part of the page and press Enter or click the "Filter" button. Only the manifestation types that contain the given input in their id or name will be displayed in the table. Filtering is case sensitive.
