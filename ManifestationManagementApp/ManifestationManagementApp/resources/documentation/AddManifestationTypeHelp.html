Adding manifestation type

When adding manifestation type, user can enter following data:

  • Type Id - Value of manifestation type identifier (id) must be unique and can be manually entered. Also, there is an option for automatic generation of id by selecting "Auto-generated Ids" field. If id is entered manually there is length constraint: from 2 to 60 characters.
  • Type Name - Represents manifestation type name. It has length constraint: from 2 to 60 characters.
  • Icon - Choose icon for manifestation type. Icon must be chosen.
  • Type Description - More detailed description about manifestation type. It has length constraint with minimum 2 characters.

By clicking on button "Add manifestation type" user adds manifestation type to system. In case of clicking on button "Cancel" adding manifestation type is cancelled and user is redirected to window that shows current manifestation types in system.
